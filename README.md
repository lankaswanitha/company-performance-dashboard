# company-performance-dashboard
company performance report
## Project Overview
This project analyzes company sales data and presents insights using an interactive Power BI dashboard.

##  Tools Used
- Power BI

## Key KPIs
- Total Sales, Total customers, Total Country Dealing, Total Categories Dealing
- Visual to show Sales and Profit Over time.
- Sales across Days of week
- Male and Female Customer Sales in each Category
- Customer Bunch Analysis by Country
- Contribution of each Category to total profit
- Contribution of company to Overall Product Line

##  Visualizations
- Bar charts
- Line charts
- Donut charts
- KPI cards
- Slicers
- Waterfall
- Heaptree
-Clustered column
